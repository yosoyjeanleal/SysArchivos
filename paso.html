Definición de Archivos en C#

En C#, los archivos son representaciones físicas de los datos almacenados en el sistema. Permiten la lectura y escritura de datos en el disco. System.IO es el espacio de nombres que se utiliza para trabajar con archivos y directorios.

Declaración de Archivos en C#

Para trabajar con archivos, es necesario utilizar el espacio de nombres System.IO. Aquí se declara un archivo utilizando las clases disponibles. 

using System.IO;

class ArchivoEjemplo {
    public void CrearArchivo() {
        string ruta = "ruta/del/archivo.txt";
        if (!File.Exists(ruta)) {
            File.Create(ruta);
        }
    }
}

Creación de Objetos para Manejo de Archivos

Para trabajar con archivos, puedes crear objetos de clases como StreamWriter y StreamReader. 

using System.IO;

class ArchivoEjemplo {
    public void EscribirArchivo() {
        string ruta = "ruta/del/archivo.txt";
        using (StreamWriter escritor = new StreamWriter(ruta)) {
            escritor.WriteLine("Este es un ejemplo de escritura en un archivo.");
        }
    }
}

Manipulación de Archivos en C#

Puedes leer el contenido de un archivo o escribir más datos en él utilizando StreamReader y StreamWriter. 

using System.IO;

class ArchivoEjemplo {
    public void LeerArchivo() {
        string ruta = "ruta/del/archivo.txt";
        using (StreamReader lector = new StreamReader(ruta)) {
            string contenido = lector.ReadToEnd();
            Console.WriteLine(contenido);
        }
    }
}

Resumen sobre Archivos en C#

El manejo de archivos en C# es fundamental para la persistencia de datos. Usando el espacio de nombres System.IO, puedes crear, leer, escribir y manipular archivos de manera eficiente.
